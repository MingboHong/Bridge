# Overview

Bridge is a basis-driven causal inference framework for domain generalization in object detection. It targets spurious correlations induced by confounders (e.g., illumination, co-occurrence, style) when training on limited single-source data. By blocking confounders through a causal mechanism, Bridge improves robustness across both discriminative and generative Vision Foundation Models (VFMs).

# Method Overview

We keep the implementation details minimal here and focus on the structure.

- Multi-scale features are extracted from a frozen VFM backbone.
- A Causal Basis Block (CBB) calibrates features via a front-door adjustment perspective in a low-rank basis space.
- Calibrated features are fed into task-specific heads for prediction.

# Conclusion

Bridge integrates causal inference into VFM-based DGOD with a lightweight, plug-and-play module. Across diverse benchmarks, varied weather, and visual styles, it consistently improves robustness and generalization. Supplemental analyses further broaden DG evaluation in remote sensing.
